### 📖 상세 정리 (Summary)
#### 1. Java 컴파일 및 실행 과정
- `Source.java` → (Compiler) → `Bytecode.class` → (JVM) → 실행
- **핵심:** OS에 종속되지 않는 독립성(WORA) 확보

#### 2. JVM 메모리 구조 (Runtime Data Areas)
- **Method Area:** 클래스 정보, 변수, static 변수 저장
- **Heap Area:** `new` 키워드로 생성된 객체와 배열이 거주 (GC의 관리 대상)
- **Stack Area:** 메서드 호출 시 생성되는 로컬 변수 및 프레임 (LIFO 구조)

#### 3. 오늘 해결한 문제 (Trouble Shooting)
- **이슈:** JDK 환경 변수 설정 후 `java -version` 미인식
- **원인:** 기존 설치된 하위 버전 JDK와 경로 충돌
- **해결:** 시스템 환경 변수 우선순위 조정 및 터미널 재시작으로 해결
